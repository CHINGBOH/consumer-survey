# 美容避坑测评小程序

## 📱 功能说明

- **首页**：介绍测评功能，展示5种类型
- **问卷页**：8道题，支持上一题/下一题
- **结果页**：显示用户类型、风险等级、核心避坑提示

## 🚀 使用方法

### 1. 安装微信开发者工具
下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `miniprogram` 文件夹
4. 填写AppID（测试可以选择"测试号"）

### 3. 预览测试
- 点击"编译"按钮
- 在模拟器中测试
- 或点击"预览"在手机上测试
